
使用 **Laravel 12** + **Vue.js + Inertia.js** 製作的 Kanban 任務管理看板，用來練習 Laravel 和 Vue 的基本使用

## 🔧 技術棧
- **Laravel 12** - 後端框架
- **Vue 3** - 前端框架
- **Inertia.js** - 前後端橋接工具
- **Vite** - 前端打包與快取重整
- **vuedraggable** - 實作拖拉排序功能
- **Axios** - 處理 AJAX 請求

## 📌 功能列表
- 任務 CRUD（新增 / 編輯 / 刪除）
- 任務拖拉
- 任務狀態即時同步更新
- 彈窗 Dialog 編輯介面
